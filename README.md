# Keras_Flask_classifier

  

## How to use this app

  

1. Type this  command in Terminal
```
$ cd Keras_Flask_classifier
```
2. Type this command in Terminal
```
$ python3 run.py
```

3. Put the link to your web browser 
*The link will show up when you execute the procedure 2
```
http://127.0.0.1:5000/
```
